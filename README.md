## Features

- Supports two user types: Riders and Drivers.
- User registration with email and phone.
- Stores hashed passwords for security.
- Supports rating for drivers.
- Supports listing available drivers.
- In-memory data storage for simplicity.

## Building

### Prerequisites

- [Java Development Kit (JDK) 21+](https://www.oracle.com/middleeast/java/technologies/downloads/)
- [Apache Maven](https://maven.apache.org/install.html)

### Setup

- Clone project repository:

```sh
git clone https://github.com/TarekSaeed0/user-management-system.git
```

- Navigate to the project directory:

```sh
cd user-management-system
```

- Build the project using Maven:

```sh
mvn compile
```
